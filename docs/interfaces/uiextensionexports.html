alert: function
  • Defined in extensions/ui/index.ts:76

Requires input from the user with a modal box.

the html to be set for the modal body.

param

buttons that the user can click to close the modal, for each item, if it is a string it will be set as the text with the default style, or the text can be specified explicitly. If no buttons are specified, a generic OK button will be used.

returns
