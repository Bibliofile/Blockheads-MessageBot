Interface IStorage

clearNamespace

clearNamespace: function
  • Defined in libraries/storage/storage.ts:55

Removes all items starting with namespace from the storage.

example

set('key+test', 1); set('key+test2', 2); clearNamespace('key+'); //both key+test and key+test2 have been removed.

param

the prefix to check for when removing items.
