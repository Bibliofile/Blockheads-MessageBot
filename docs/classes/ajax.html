Static get

  • get(url?: string, params?: object): Promise<string>

    Function to GET a page. Passes the response of the XHR in the resolve promise.

    example

    get('/some/url.php', {a: 'test'}).then(console.log) //sends a GET request to /some/url.php?a=test

    • Default value url: string = "/"

      the url to fetch.

    • Default value params: object = {}

      any parameters to add to the URL in the format of key=value
